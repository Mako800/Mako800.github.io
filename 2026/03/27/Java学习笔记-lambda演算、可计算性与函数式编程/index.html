学校老师“制作精良”的PPT和“逻辑严谨”的讲课风格，使我决定写下这样一篇笔记，以梳理我对于可计算性理论相关问题的一点浅薄的理解。

λ\lambdaλ演算是一个看上去非常无厘头，但理解之后让人惊叹于发明者的天才构想的一种语言。它可以说是在某种程度上解释了“计算”这一行为的本质。或者干脆说，λ\lambdaλ演算本身就是一门编程语言。

引论

我们先来回想一下数学中的“函数”这一概念，我们可以定义一个函数，例如：

f(x,y)=x+2yf(x,y)=x+2y f(x,y)=x+2y

这个函数描述了一种映射，即把二维平面上的一个点映射到一个实数。所以，事实上我们并不关心函数名字是什么。你可以叫他fff，也可以叫做哈基米，或者叫做MyGo!!!!!MyGo!!!!!MyGo!!!!!也没有问题。为了避免名称所带来的限制，我们可以把它写作这样：

(x,y)→x+2y(x,y)\to x+2y (x,y)→x+2y

注意到了吗？描述这个函数，我们只需要三个部分：

  1. 变量（即这里面的 x,yx,yx,y）
  2. 函数标识（这里面是那个箭头，表明这是个映射）
  3. 映射规则（既函数体 x+2yx+2yx+2y）

所以，我们做一下简化。

λ\lambdaλ演算的定义

λ\lambdaλ演算最为严谨的定义是基于 λ 项的，不过那个太难理解。这里我给出一种并不严谨但便于理解的定义。

单变量的λ\lambdaλ演算

形如 λx.f(x)\lambda x.f(x)λx.f(x) 的式子是一个 λ 演算。其中，λ\lambdaλ 到 ... 中间的部分是声明变量名，点号是区分声明与函数体的间隔符，f(x)f(x)f(x) 是函数体，描述对 xxx 的某种操作。

例如，以下算式都是 λ 演算：

λx.x\lambda x.x λx.x

λa.a\lambda a.a λa.a

λx.x  x\lambda x.x\;x λx.xx

你应该已经注意到了，这样只能声明一个变量，我们能做的事情还很有限。那么，能不能定义多变量函数呢？答案是肯定的，这就要清楚我们的柯里化（Currying）。

柯里化

形如 λx1.λx2.  ...  λxn.f(x1,x2,...,xn)\lambda x_1.\lambda x_2.\; ...\; \lambda x_n.f(x_1,x_2,...,x_n)λx1​.λx2​....λxn​.f(x1​,x2​,...,xn​) 的式子是一个 λ 演算。其中，第一个 λ\lambdaλ 到最后一个 ... 中间的部分是声明变量名，点号是区分声明与函数体的间隔符，f(x1,x2,...,xn)f(x_1,x_2,...,x_n)f(x1​,x2​,...,xn​) 是函数体，描述对 xix_ixi​ 的某种操作。

例如，以下算式都是柯里化的多变量 λ 演算：

λx.λy.x  y\lambda x.\lambda y.x\;y λx.λy.xy

λa.λb.a  b\lambda a.\lambda b.a\;b λa.λb.ab

λx.λy.x+2y\lambda x.\lambda y.x+2y λx.λy.x+2y

你应该已经注意到了，上面的 λx.λy.x  y\lambda x.\lambda y.x\;yλx.λy.xy 和 λa.λb.a  b\lambda a.\lambda b.a\;bλa.λb.ab 其实没有本质区别，知识声明的函数变量名不同。那么，这就要提到我们的 α\alphaα等价了。

α\alphaα等价

自由变量

不被声明的变量是自由变量，例如 λx.(x  y)\lambda x.(x\;y)λx.(xy) 中，xxx 是非自由变量，yyy 是自由变量。

α\alphaα等价的操作

把一个λ 演算表达式中所有同名的非自由变量\boldsymbol{非自由变量}非自由变量变换为相同的另一个字母，就是α\alphaα等价。它不改变算式的含义。例如：

λx.λy.x+2y\lambda x.\lambda y.x+2y λx.λy.x+2y

α\alphaα等价于

λa.λb.a+2b\lambda a.\lambda b.a+2b λa.λb.a+2b

而

λx.λy.x  y\lambda x.\lambda y.x\;y λx.λy.xy

α\alphaα等价于

λa.λb.a  b\lambda a.\lambda b.a\;b λa.λb.ab

现在，你或许产生了一些疑惑：刚才的 x+2yx+2yx+2y我可以理解，可是直接写一行 x  yx\;yxy，这是什么意思呢？

β\betaβ规约

应用

函数是可以带入具体的值的，λ\lambdaλ 演算当然也可以。而且，代入的并不一定是“值”，还可以是其他的一些东西，例如变量，甚至是另一个函数。
