# 什么是模拟退火
　　简单来说，就是模拟物理学意义上温度逐渐降低的过程中，系统中的一个分子的运动。它是一种随机化算法，用于~~乱搞~~求多峰函数最大或最小值。
# 瞎子爬山
　　试想，如果给你一个单峰函数，怎么求其最大值？
  
　　先随机一个起始点，然后试探一个新的点，如果更优就走过去，否则舍弃。
  
　　应该遵循的原则是：

　　　　1. 刚开始试探的步子应该大一些，越到后期步子应越小，不然就跑乱了。

　　　　2. 每次试探到的新位置在原有位置基础上随机。


　　但是显然，瞎子爬山如果遇到了多峰函数，就GG。因为有可能走到的“山峰”是局部最大值而不是全局最大值。
# 模拟退火
### 算法思想
　　模拟退火是对瞎子爬山的改进，思想如下：
  
　　先随机一个起始点，然后试探一个新的点，如果更优就走过去，否则**以一定概率接受这个较差的解**。
   
　　可以看出，模拟退火与瞎子爬山的区别在于黑体加粗的文字。这样使得算法有机会跳出局部最优解，有更大可能性找到全局最优解。
   
　　那这个概率应该是多少呢？经过一系列~~我也不会的~~数学物理上的计算得出结论：

　　　　　　　　$ p = e^{-d/T} , d<0 $

　　其中，p为接受较差解的概率，e为自然对数的底，T为当前温度，d为新答案价值-原答案价值（注意，顺序不能反）。
### 算法步骤
  1. 初始化一个温度 $T_0$，降温系数 $\alpha$，初始位置 $x_0$，还需要一个计算当前状态价值的函数 $cal(s)$。
  2. 如果 $T_0>= end$ , 就找到一个新变量 $x_1=x_0+rand()*T_0 $ ，若 $cal(x_1)>cal(x_0)$，则令$x_0=x_1$，否则，令$d=cal(x_1)-cal(x_0)$，若 $exp(-d/T_0)*$RAND_MAX$>rand()$，则令$x_1=x_0+rand()*T_0 $。
  3. 重复步骤二2，每次结束后要降温，即令$T_0 = T_0 * \alpha$，直到 $T_0<end$。


### 算法实现
```cpp
double ans,energy;     // ans表示最终答案，energy表示当前状态下的价值
void san()
{
	double t=3000;         // 初始温度
	double delta=0.997;    // 降温系数
	while(t>1e-15)     // 温度大于某个很小的值就一直执行
	{
		double new_ans=ans+rand()*t;             // 随机一个新位置
		double new_energy=cal_energy(new_ans);   // 计算新位置的价值
		double de=new_energy-energy;             // 新旧位置的价值差值
		if(de>0)   // 如果新位置更优，直接更新
		{
			ans=new_ans;
			energy=new_energy;
		}
		else if(exp(-de/t)*RAND_MAX>rand())    // 否则以一定概率更新
		{
			ans=new_ans;
		}
		t*=delta;   // 降温
	}
}
```
